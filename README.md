Para colocar em prática os conceitos aprendidos sobre o consumo de APIs, desenvolvi um projeto de site simples para verificação de clima. Este projeto foi construído utilizando as seguintes tecnologias:

JavaScript​: Responsável pela lógica de consumo da API externa de clima, manipulação dos dados recebidos (JSON) e atualização dinâmica da interface do usuário.

HTML: Estruturação da página web, definindo os elementos onde as informações climáticas seriam exibidas.

CSS: Estilização do site, garantindo uma apresentação visual agradável e responsiva.
